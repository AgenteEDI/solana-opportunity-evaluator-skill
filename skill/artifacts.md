# Output Artifacts

## Standard memo

Use this shape:

```text
Opportunity:
Type:
Why it exists:

Facts:
- ...

Scores:
- strategic_fit:
- execution_fit:
- upside:
- competition_position:
- time_sensitivity:
- downside_risk:

Verdict:
- pursue / defer / drop

Why:
- ...

Open questions:
- ...

Recommended next step:
- ...
```

## Short verdict format

Use this when the user wants speed:

```text
Verdict: pursue / defer / drop
Main reason:
Main risk:
Next move:
```

## Comparison mode

When comparing multiple opportunities:

- rank them
- explain what makes the winner better
- explain why the others lose
- state which one compounds into future assets
